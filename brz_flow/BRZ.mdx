# BRZ Token na Flow

O BRZ é a principal stablecoin brasileira na blockchain Flow, mantendo paridade 1:1 com o Real Brasileiro (BRL).

## Características

- **Pareamento 1:1 com Real Brasileiro**
- **Totalmente lastreada e auditada**
- **Compatível com padrão FungibleToken da Flow**
- **Suporte multi-chain**

## Como Funciona

O BRZ na Flow opera através de um smart contract que segue o padrão FungibleToken, permitindo:

- **Transferências entre contas**
- **Integração com dApps**
- **Consulta de saldos**
- **Operações de mint e burn controladas**

## Casos de Uso

### DeFi

- **Empréstimos**
- **Yield farming**
- **Pools de liquidez**
- **Staking**

### Pagamentos

- **Transferências P2P**
- **Integrações comerciais**
- **Remessas internacionais**
- **Pagamentos em dApps**

## Começando

```cadence
import FungibleToken from 0xf233dcee88fe0abe
import BRZ from 0x123456789 // endereço do contrato BRZ

transaction(amount: UFix64, recipient: Address) {
    prepare(signer: AuthAccount) {
        let payment <- signer.borrow<&BRZ.Vault>(from: /storage/brzVault)!
            .withdraw(amount: amount)

        let recipientAccount = getAccount(recipient)
        let receiver = recipientAccount
            .getCapability(/public/brzReceiver)
            .borrow<&BRZ.Vault{FungibleToken.Receiver}>()
            ?? panic("Recipient not set up to receive BRZ")

        receiver.deposit(from: <-payment)
    }
}
```
